# provejs-params

Prove function parameters using a simple schema.

## Install

```bash
npm install provejs-params --save
```

## Usage

```js
var Prove = require('provejs-params');

function myFunction(myArray, myNumber) {
	Prove('AN', arguments);
}

myFunction([], []); // bug here
```
